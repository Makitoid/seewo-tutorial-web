---
title: 网络连接问题
---

稳定的网络连接是访问在线教学资源的基础。网络问题通常分为有线网络和无线网络两类。

## 有线网络连接问题

### 无法识别网线（显示“网络电缆被拔出”）
*   **现象**：插入网线后，系统图标显示红叉或提示“网络电缆被拔出”。
*   **排查步骤**：
    1.  **检查物理连接**：确保网线两端（一体机网口和墙上面板/交换机网口）均已插紧，听到“咔哒”声。尝试更换一根确认正常的网线。
    2.  **检查接口指示灯**：观察一体机网口旁的指示灯是否亮起。如果指示灯不亮，可能是端口故障或对端设备（如交换机）未供电或故障。
    3.  **驱动程序**：在“设备管理器”中检查“网络适配器”下的有线网卡驱动是否正常，有无黄色叹号。可尝试禁用后再启用该设备，或更新驱动。

### 已连接但无法上网（显示“无Internet”）
*   **现象**：网络连接显示已识别，但浏览器无法打开网页。
*   **排查步骤**：
    1.  **获取IP地址**：按 `Win + R` 键，输入 `cmd` 并回车，在命令提示符中输入 `ipconfig`。查看是否获得了以 `192.168.x.x` 或 `10.x.x.x` 开头的有效IP地址。如果IP地址以 `169.254` 开头，说明未能从学校路由器获取到IP地址，可能是网络配置或DHCP服务问题，需联系网络管理员。
    2.  **DNS问题**：尝试在命令提示符中运行 `ipconfig /flushdns` 清除DNS缓存。也可以尝试将DNS服务器手动设置为公共DNS，如 `114.114.114.114` 或 `8.8.8.8`。
    3.  **防火墙或代理**：检查是否误设了代理服务器（在Internet选项中查看），或暂时关闭防火墙测试。

## 无线网络（Wi-Fi）连接问题

### 搜索不到或无法连接Wi-Fi
*   **现象**：无法搜索到学校的Wi-Fi信号，或反复连接失败。
*   **排查步骤**：
    1.  **开启无线功能**：确认一体机的无线功能已开启（有些有物理开关或快捷键 `Fn + F2` 等）。
    2.  **检查信号强度**：将一体机移动到离路由器稍近的位置，或调整一体机天线的角度（如果可见）。
    3.  **忘记网络后重连**：在Wi-Fi设置中，找到学校的网络，选择“忘记”，然后重新输入密码连接。
    4.  **驱动问题**：同有线网络，检查并更新无线网卡驱动。

### Wi-Fi信号弱、网速慢或频繁断线
*   **现象**：虽然已连接，但信号格数少，上网速度很慢，或不稳定。
*   **排查步骤**：
    1.  **减少干扰**：一体机通常靠墙安装，金属黑板会对Wi-Fi信号产生严重屏蔽。尽量让天线远离金属物体。检查周围是否有微波炉等强干扰源。
    2.  **切换频段**：如果学校同时提供2.4GHz和5GHz的Wi-Fi，尝试连接5GHz频段，通常干扰更少，速度更快（但穿墙能力稍弱）。
    3.  **电源管理**：在“设备管理器”中右键点击无线网卡，进入“属性”->“电源管理”，取消勾选“允许计算机关闭此设备以节约电源”。

## 通用排查方法

1.  **重启设备**：重启一体机和路由器是解决大部分临时性网络故障的首选方法。
2.  **使用网络疑难解答**：Windows自带的网络疑难解答功能（在网络设置中）有时能自动发现并修复问题。
3.  **ping命令测试**：在命令提示符中，先 `ping 127.0.0.1` 测试本机网络协议是否正常，再 `ping` 一个外网地址（如 `ping 114.114.114.114`），判断网络连通性。

> **提示**：如果教室中只有这一台设备无法上网，多为本机设置问题。如果多台设备同时出现故障，则很可能是校园网络或路由器的问题，需联系网络管理员。
