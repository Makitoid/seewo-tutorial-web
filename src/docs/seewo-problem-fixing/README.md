---
title: 一体机故障修复
---

在日常使用一体机进行教学或课余活动时，难免会遇到一些设备问题。本节汇总了常见的故障现象及其排查思路，希望能帮助您快速解决问题，保障教学顺利进行。

- [开机与启动问题](startup-issues.md)：设备无法通电、开机无显示或无法进入系统。
- [显示与触摸问题](display-touch-issues.md)：屏幕无信号、显示异常、触摸失灵或不准。
- [声音问题](audio-issues.md)：设备没有声音、音量过小或存在杂音。
- [网络连接问题](network-issues.md)：无法连接Wi-Fi、有线网络不稳定或网速慢。
- [软件与系统问题](software-system-issues.md)：系统卡顿、软件无法打开或文件丢失。
- [外设连接问题](peripheral-issues.md)：U盘无法识别、遥控器失灵或其他外接设备问题。
- [硬件维护与清洁](hardware-cleaning.md)：设备清灰、屏幕清洁及日常保养建议。
