如果说硬件是一体机的身体，那么软件就是它的灵魂。优秀的软件能让一体机的日常使用体验更加流畅顺手，有效提升教学效率。

- [操作系统推荐](./operating-system.md)：软件生态的核心，优秀用户体验的基础。
- [核心工具推荐](./core-utilities.md)：增强系统核心功能，优化日常使用体验。
- [演示软件推荐](./presentation-apps.md)：教学场景的重点，学生知识获取的窗口。

> 本节仅介绍课堂所需的基础软件。如需探索更多适合教学场景的软件，可参考 [Awesome IWB 项目](https://github.com/Awesome-Iwb/Awesome-Iwb) 或 [ACW 项目](https://github.com/Jursin/Awesome-Class-Softwares)。
