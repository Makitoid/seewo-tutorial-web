解压缩、文件搜索、软件卸载等基本操作在一体机使用中十分常见。工欲善其事，必先利其器。高效的工具能大幅提升日常使用体验。

## 解压缩软件

推荐使用 **Bandizip**。它是一款轻量、快速且功能强大的压缩软件，支持包括 RAR、ZIP、7Z 在内的绝大多数压缩格式。其免费版本界面清爽，无任何广告，解压速度极快，并提供了独特的“自动解压”功能，能有效避免解压文件散落一桌面的情况。

[**PeaZip**（点击跳转GitHub仓库）](链接3) 也是一个优秀的开源跨平台选择，注重安全性和格式支持，界面简洁大方。

## 文件搜索工具

**Everything** 是由 voidtools 开发的一款文件名搜索软件，堪称 Windows 系统自带搜索功能的“终极加速器”。它能在秒级内索引并搜索您电脑上的所有文件和文件夹，输入关键词的瞬间即可呈现结果，极大提升了文件查找效率。该软件体积小巧，资源占用极低。

## 软件卸载工具

推荐使用 **Geek Uninstaller**。当使用 Windows 自带的“添加/删除程序”卸载软件后，往往会有残留的文件和注册表项。Geek Uninstaller 是一款轻量级的免费软件，可以彻底卸载应用程序并扫描清除所有残留的垃圾，确保系统清洁。

## 还原软件

推荐使用冰点还原（Deep Freeze）。先启动 `setup` 进行安装，同意用户协议，安装进入到许可证密钥界面不填写，打钩使用评估版。选择冻结的驱动器配置时只选择C盘，勾选“保持新发现的硬盘驱动器为解冻状态”，之后一路下一步进行安装。之后电脑会自动重启一次，重启之后安装完成。

重启后，按 <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Alt</kbd>+<kbd>F6</kbd> 唤出软件，设置密码后，选择【启动后解冻】，然后保存重启；打开C盘，找到【Persi0.sys】，用 360 安全卫士解除该软件的占用然后运行破解补丁，点击【Patch】按钮，等待弹出提示框，点击【OK】，然后点击【Exit】退出即可完成破解。

破解后再次按下 <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Alt</kbd>+<kbd>F6</kbd> 重新打开冰点还原，选择【启动后冻结】，点击确定重启电脑即可。

## 视频播放器

推荐使用 **PotPlayer**。它可以解码目前市面上几乎所有的视频和音频格式，功能强大，且对触摸屏有良好优化。缺点是有新版本时默认会弹出升级提醒，建议在设置还原之前关闭自动更新。可在播放界面右键 →【选项】→【基本】→【自动更新】→【不使用自动更新】以关闭。

## 倒计时软件

推荐使用 mamsdn 桌面倒计时软件。其倒计时样式调整自由度高，可以满足大部分场景的需求。注意首次开启时可能会弹出 2345 网址导航的弹窗，关闭即可。

## Github 加速工具

众所周知，GitHub 在国内的访问时常不稳定，而许多教学辅助软件（如 Classisland, ica）都依赖其进行更新。虽然在班级电脑上安装 VPN 软件可能不太合适，但可以使用专门的加速工具。

[**Watt Toolkit**（原名 Steam++，点击跳转GitHub仓库）](链接2) 是一个开源跨平台的多功能工具箱。虽然其开发初衷是针对 Steam 平台，但其内置的“网络加速”功能对 GitHub、谷歌学术等常见开发和研究网站有很好的加速效果。在 3.0 及以上版本中，您可以禁用其他不需要的插件，只保留核心的加速功能。
